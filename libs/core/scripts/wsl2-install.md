# WSL2

https://docs.microsoft.com/en-us/windows/wsl/install-win10

## Install WSL2

```bash
wsl --install`
```

## Update WSL kernel

```bash
wsl --update
wsl --shutdown
```

## Install your Linux distributions of choice

**Ubuntu 18.04 LTS -> Install**

```bash
wsl --install -d Ubuntu-18.04
```

**Ubuntu 20.04 LTS -> Install**

```bash
wsl --install -d Ubuntu-20.04`
```
